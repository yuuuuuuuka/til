② コントローラーを変更する

## ①Formクラスを作成しよう
最初にFormクラスを作成します。Formクラスとは、フォームに入力されたデータを格納するためのクラスです。


railsでは、paramsというハッシュによって自動で保存ができました。
それに代わるものとして、javaのFormクラスは自前で作成します。
