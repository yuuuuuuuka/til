# 一覧表示機能を実装

Javaでデータベースを利用することは少し難しいため、以下の流れで実装を行なっていきます。


① データベースは使用せず、コントローラー内で作成したリストをビューで表示できるようにする


② データベースと連携し、テーブル内のデータをビューで一覧表示できるようにする



FirstAppでは、以下のようにidカラムとmemoカラムのみを扱うシンプルなテーブルを作成します。

id: 投稿ごとにつけるid

memo：	投稿されたテキスト
